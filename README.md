# Название проекта
"Система записи звука"

# Описание проекта
Проект представляет собой управляемое устройство для записи и обработки звука.

# Используемые технологии
Аппаратная часть:
- Микроконтроллер ESP32
- Микрофон MAX9814
- Микрофон INMP441
- SD картридер

Программная часть:
- Python
- ...
